# MS-Sentinel-Cybersecurity-SIEM-Demo
This is an overview demonstration of my Microsoft Sentinel SIEM lab setup with ingesting logs information, parsing out data for analysis via KQL language and displaying using a simple dashboard of log attempts attacks.

In this lab demo - using Microsoft Azure VM as a vector - I created several resources and a MS sentinel log source - opened ports to internet exposing log in attempts from external sources - upon creating an attack vector and log filtering - I noticed several attempts from multiple locations which is displayed on my location dashboard.
